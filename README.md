# MovieGenerator

## Summary
##### This is a Full-stack CRUD web app that allows users to look up movie recommendations, and add them to a watch list that can be accessed when they login to their own profiles. Users can also add a short review to their favorite movies that will help other users discover new favorites.

### Project Group Members
#### Luke Fariselli, Tylaine Jeffrey, Jonathan Lora, Kendrick Zhang.


## Frameworks Used
##### Javascript ES6, Node.js, Express.js, React, PostgreSQL, CSS, HTML.

## Dependencies Used
##### PG-Promise, Morgan, Body-Parser, Nodemon, Method-Override, CORS

## API Used
##### [TMDB API](https://www.themoviedb.org/documentation/api)


## User Stories
##### - As a user I want to be given recommendations of what not to miss in pop culture.

##### - As a user I want to be given daily recommendations because i can't decide on my own.

##### - As a user I want to be able to create a watchlist of things i need to or want to watch.

##### - As a user I want to be able to write reviews and help others decide what to watch.

##### - As a user I want to have custom recommendations.
